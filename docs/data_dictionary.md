# Data Dictionary

## Mart layer

Одна строка = агрегированные температурные показатели за один день по одному городу.

| column | бизнес-смысл | единица | примечание |
|--------|-------------|--------|-----------|
| date | дата наблюдения | date | timezone Asia/Novosibirsk |
| city_id | уникальный идентификатор города | - | ключ |
| city_name | название города | - | |
| country_code | код страны | - | |
| temperature_mean | средняя температура за день | °C | среднее по часам |
| temperature_min | минимальная температура за день | °C | |
| temperature_max | максимальная температура за день | °C | |
| temperature_range | размах температуры | °C | max - min |
